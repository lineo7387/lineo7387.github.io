# Typography as a Design Element

In Bauhaus design, typography was not merely a vehicle for words—it was a primary visual element. Letters became shapes, and text became texture.

## The Bauhaus Typography Revolution

Before Bauhaus, typography was constrained by historical conventions. The Bauhaus movement broke these rules:

- **Sans-serif dominance** — Clean, geometric letterforms replaced ornate serifs
- **Experimental layouts** — Text was rotated, stacked, and arranged dynamically
- **Functional hierarchy** — Typography clearly communicated importance

## Choosing the Right Typeface

For Bauhaus-inspired design, select typefaces that embody geometric principles:

### Characteristics to Seek

- Circular letterforms (o, e, c)
- Consistent stroke weights
- Clean terminals without decorative flourishes
- Excellent readability at all sizes

### Recommended Fonts

- **Outfit** — Modern geometric sans-serif with excellent versatility
- **Space Grotesk** — Technical feel with distinctive character
- **DM Sans** — Warm geometric with humanist touches

## Typography Scale and Hierarchy

Create visual hierarchy through size contrast:

```
Display: 4xl - 8xl (headlines)
Heading: 2xl - 4xl (section titles)
Subheading: xl - 2xl (paragraph headers)
Body: base - lg (paragraphs)
Caption: sm - xs (metadata)
```

## The Power of Uppercase

Bauhaus typography embraces uppercase for impact:

- Headlines in uppercase create authority
- Labels and navigation benefit from uppercase clarity
- Combine uppercase headers with sentence case body text

## Letter Spacing and Line Height

### Tracking (Letter Spacing)

- **Headlines**: `tracking-tighter` or `tracking-none` — letters should breathe
- **Body**: Default spacing for readability
- **Labels**: `tracking-wider` or `tracking-widest` — creates visual distinction

### Line Height

- **Headlines**: `leading-[0.9]` — tight for impact
- **Body**: `leading-relaxed` (1.6+) — comfortable reading
- **UI Elements**: `leading-normal` — compact for buttons and labels

## Practical Tips

1. Limit to 2-3 typefaces maximum
2. Create hierarchy through size, weight, and case—not color
3. Test readability on actual devices
4. Use generous line height for long-form content
5. Align text consistently (left is usually best)
